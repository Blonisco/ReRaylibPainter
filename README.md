# ReRaylibpainter

## About
这是关于RaylibPainter的一个重写版本，使其分文件编写，更加易于管理  
这主要是北邮的记导实验作业。
主要在NeoVim下完成。

## 功能
·必需要从Normal模式下进入各个绘画模式  
按tab返回Normal模式（你如果会使用Vim的话，应该会熟悉这操作，但切记不要按esc！）  
·按下C键进入Circle模式绘制圆    
·按下L键进入Line模式绘制线段  
·按下T键进入Text模式  
·按下P键进入Poly模式绘制多边形  
按下鼠标绘制顶点，按下Enter完成单个多边形绘制  
请注意，保持多边形为凸多边形，否侧图像不匹配  
再请注意，必须按逆时针绘制顶点！  
·滚轮调节大小
·按数字键调节颜色  
·按U撤回操作
