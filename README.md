# ReRaylibpainter

## About
这是关于RaylibPainter的一个重写版本，使其分文件编写，更加易于管理  
这主要是北邮的记导实验作业。
主要在NeoVim下完成

## 功能
·按下C键进入Circle模式绘制圆    
方向上下键调节圆大小   
·按下L键进入Line模式绘制线段  
·按下P键进入Poly模式绘制多边形  
按下鼠标绘制顶点，按下Enter完成单个多边形绘制  
请注意，保持多边形为凸多边形，否侧图像不匹配  
·按数字键调节颜色  
·按U撤回操作

## 编译源代码
·Windows下直接命令行运行make即可
·Linux下把makefile那里的注释#去掉在删除上一行应该可以，（未测试）
